# Quickstart

1. Copy env template:
   - `cp .env.example .env.local`
2. Install dependencies:
   - `yarn install`
3. Start dev server:
   - `yarn dev`
4. Run quality checks:
   - `yarn check:all`
5. Build UI with project primitives:
   - Prefer `src/components/ui` shadcn/ui components for new screens and features.
